# 项目概览

---

## 1. 项目整体结构

```
main.cpp → 创建窗口
MainWindow → 界面 + 协调各模块
NetworkManager → 管理网络通信（TCP/UDP）
SerialPortManager → 管理串口通信
DataProcessor → 处理数据格式
LogManager → 日志记录
```

## 3. 关键 Qt 类

| 类 | 作用 |
|---|------|
| `QApplication` | 应用程序主类，管理事件循环 |
| `QMainWindow` | 主窗口类，有菜单栏、状态栏等 |
| `QTcpSocket` | TCP 客户端 Socket |
| `QTcpServer` | TCP 服务器，监听连接 |
| `QByteArray` | 字节数组，用于存储二进制数据 |
| `QTextEdit` | 多行文本编辑框 |
| `QLineEdit` | 单行文本输入框 |
| `QPushButton` | 按钮 |
| `QComboBox` | 下拉选择框 |
| `QVBoxLayout` | 垂直布局 |
| `QHBoxLayout` | 水平布局 |
| `QGridLayout` | 网格布局 |
