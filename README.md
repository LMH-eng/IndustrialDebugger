# 工业级网络/串口调试助手

一款基于Qt5开发的跨平台工业级调试工具，支持TCP/UDP网络通信和串口通信，专为工业设备调试场景设计。

## 功能特性

### 核心功能
- ✅ **TCP客户端/服务端**：支持TCP双向通信，自动处理粘包问题
- ✅ **UDP通信**：支持UDP单播数据收发
- ✅ **串口通信**：支持波特率、数据位、校验位等全参数配置
- ✅ **数据格式转换**：ASCII/十六进制格式一键切换
- ✅ **时间戳标记**：自动添加收发时间戳
- ✅ **日志管理**：支持日志导出/导入功能
- ✅ **收发统计**：实时统计收发数据量

## 快速开始

### 环境要求

- Qt 5.15.2 或更高版本
- CMake 3.16 或更高版本
- C++17 支持的编译器

### 编译步骤

#### Windows (MinGW)

```bash
# 1. 创建构建目录
mkdir build
cd build

# 2. 配置CMake
cmake -G "MinGW Makefiles" -DCMAKE_PREFIX_PATH="C:/Qt/5.15.2/mingw81_64" ..

# 3. 编译
mingw32-make

# 4. 运行
./bin/IndustrialDebugger.exe
```

#### Linux

```bash
# 1. 创建构建目录
mkdir build
cd build

# 2. 配置CMake
cmake -DCMAKE_PREFIX_PATH="/usr/lib/qt5" ..

# 3. 编译
make

# 4. 运行
./bin/IndustrialDebugger
```

### 使用说明

1. **网络通信**
   - 选择TCP客户端/服务端或UDP模式
   - 配置IP地址和端口
   - 点击"连接"按钮开始通信

2. **串口通信**
   - 选择串口设备
   - 配置波特率、数据位、校验位、停止位
   - 点击"打开串口"开始通信

3. **数据收发**
   - 在发送框输入数据
   - 选择ASCII或Hex格式
   - 点击"发送"按钮
   - 接收的数据会显示在日志区
